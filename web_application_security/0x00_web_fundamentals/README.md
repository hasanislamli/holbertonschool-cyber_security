# Web Fundamentals

## Description
This project is part of the Holberton School Cyber Security curriculum.

The objective is to understand how web applications work, including:
- HTTP and HTTPS
- Front-end and Back-end communication
- Stateful vs Stateless protocols
- Structured vs Unstructured data
- OWASP Top 10
- SQL Injection
- Cross-Site Scripting (XSS)
- Bug Bounty concepts

## Environment
- Kali Linux 2023.3
- curl
- sqlmap
