# 0x01. OWASP Top 10

## Description

This project covers the **OWASP Top 10 (2021)**, the most critical web application security risks. The tasks focus on understanding common vulnerabilities and implementing simple security-related scripts using Bash.

## Learning Objectives

At the end of this project, you should be able to explain:

* What OWASP is
* The OWASP Top 10 (2021) categories
* Common web application vulnerabilities
* Cryptographic Failures
* Injection attacks
* Security Misconfiguration
* Server-Side Request Forgery (SSRF)
* Identification and Authentication Failures
* Software and Data Integrity Failures
* Security Logging and Monitoring Failures
* Basic Bash scripting for security tasks

## Requirements

* Kali Linux 2025.4
* Bash
* Betty style
* All scripts must begin with:

```bash
#!/bin/bash
```

* All scripts must end with a new line.
* No use of backticks, `&&`, `||`, or `;`.
