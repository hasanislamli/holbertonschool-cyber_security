# Network Security - Active Reconnaissance

## Description
This project focuses on the core principles of **Active Reconnaissance** in cyber security. Unlike passive reconnaissance, active reconnaissance involves direct interaction with the target system to gather critical infrastructure details, identify open ports, map network topologies, and discover potential vulnerabilities.

The exercises in this module are designed to build a logical framework for penetration testing, utilizing industry-standard tools such as `Nmap`, `Nikto`, `Masscan`, `Gobuster`, and `SQLMap`.

## Technologies & Tools
* **OS:** Kali Linux
* **Networking:** OpenVPN (Lab Environment Connection)
* **Tools:** Nmap (Network Mapper)

## Learning Objectives
By the end of this project, you will be able to explain:
* The definition and implications of active reconnaissance.
* How to efficiently scan a target machine using various `Nmap` techniques.
* The difference between network mapping and web vulnerability scanning.
* The importance of understanding open doors (ports) before attempting exploitation.
